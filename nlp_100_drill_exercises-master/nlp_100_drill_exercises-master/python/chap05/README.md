## Chương 5: Dependency parsing (係り受け解析)
Thực hiện phân tích cấu trúc ngữ pháp (dependency parsing) bằng công cụ [CaboCha](http://taku910.github.io/cabocha/) cho file [neko.txt](http://www.cl.ecei.tohoku.ac.jp/nlp100/data/neko.txt) và lưu kết quả vào file neko.txt.cabocha. Sử dụng file kết quả này làm đầu vào cho các bài tập dưới đây.

### Thực hiện phân tích cấu trúc ngữ pháp cho file neko.txt
    cabocha -f1 ../../data/neko.txt -o ../../data/neko.txt.cabocha
